This is a web application to analyse different data of the series Game of Thrones per episode. The data goes up to and including the 5th season.

The features which can be used are the following:

  • Show information about characters
  • Show relationships between different characters
  • Show number of deaths per episode
  • Show number of TV viewers per episode

Show information about characters

  • Hover over one of the circles to display a character's name.
  • Clicking on a circle displays all relation information about a character in the big info box

Show the relationship between different characters.

  • Clicking on one of the circles toggles it's relaitonships on and off. If you don't see any relations, it could be that character hasn't appeared in the show yet or really has no active relationships in that episode. Furthermore, if you still see relationsship lines on a circle you just switched off, it means the other person this relation belongs to is still acitve.
  • It is also possible to display the relationship of all characters to an episode by clicking on the "draw all relations" button.
  • In order to delete all drawn relationship you can click on the button "clear relations".
  • Using the slider you can control the current episode determining which relations are displayed. You can see the current episode in the small info box.

Number of deaths per episode

While changing the episode with the slider, the number of deaths for this episode will be displayed in the info box.

Number of TV-Viewers per episode

While changing the episode with the slider, the number of TV-Viewers(US) at the first broadcast for this episode will be displayed in the info box.
